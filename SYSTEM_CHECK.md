# 🔍 Проверка системы перед демонстрацией

## ✅ Checklist для запуска

### 1. Файлы и зависимости
- [ ] `.env` файл создан с `BOT_TOKEN`
- [ ] `python-dotenv` установлен (в requirements.txt)
- [ ] `aiohttp` установлен (в requirements.txt)
- [ ] Все файлы на месте

### 2. Компоненты системы
- [ ] `monitors/whale_monitor.py` - хранение алертов добавлено
- [ ] `bot.py` - HTTP API сервер добавлен
- [ ] `handlers/webapp.py` - команда /dashboard создана
- [ ] `static/miniapp.html` - Mini App готова

### 3. Тестирование команд
```bash
# Запуск бота
python bot.py

# Проверка команд в Telegram:
/start          # приветствие
/demo            # демо-алерты
/status          # статус системы
/dashboard       # кнопка Mini App
/test            # тестовый алерт
```

### 4. Проверка API
```bash
# После запуска бота:
curl http://localhost:8080/api/transactions
```

Ожидаемый результат: `[]` или JSON с транзакциями

## 🚨 Если что-то не работает

### Бот не запускается:
```bash
# Проверьте токен в .env
echo $BOT_TOKEN

# Проверьте зависимости
pip install -r requirements.txt
```

### API не отвечает:
```bash
# Проверьте порт
netstat -an | grep 8080

# Проверьте логи
tail -f logs/bot.log
```

### Mini App не загружается:
```bash
# Проверьте ngrok
curl https://ваш-ngrok.ngrok.io/api/transactions

# Проверьте GitHub Pages
# Откройте URL в браузере
```
