# 🚀 Быстрый запуск Whale Tracker Bot

## Минимальная настройка (2 минуты)

### 1. Установка зависимостей
```bash
pip install -r requirements.txt
```

### 2. Создание .env файла
```bash
# Скопируйте шаблон
cp .env.example .env

# Откройте .env и вставьте только BOT_TOKEN:
BOT_TOKEN=123456789:ABCdefGHIjklMNOpqrsTUVwxyz
```

### 3. Запуск бота
```bash
python bot.py
```

## Где взять BOT_TOKEN?

1. Откройте Telegram
2. Найдите **@BotFather**
3. Отправьте команду `/newbot`
4. Введите имя бота: `Whale Tracker Bot`
5. Введите username: `whale_tracker_bot`
6. Скопируйте полученный токен

## Проверка работы

После запуска отправьте боту:
- `/start` - приветствие
- `/demo` - демонстрация алертов
- `/status` - статус и дашборд

## Демо-режим

Бот работает в демо-режиме по умолчанию, поэтому:
- ✅ Не нужны API ключи
- ✅ Не нужна настройка
- ✅ Алерты приходят сразу

## Если что-то не работает

1. Проверьте BOT_TOKEN в .env
2. Убедитесь что python-telegram-bot установлен
3. Проверьте права доступа к файлам

**Готово! Бот работает и готов к демонстрации клиентам!** 🎯
